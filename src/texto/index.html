Texto

Texto CSS es el módulo de CSS que define cómo realizar la manipulación de elementos de texto como los saltos de línea, la justificación, la alineación, la gestión de espacios en blanco y las transformaciones de texto.

En CSS, la propiedad font-family define un tipo de letra específico para un elemento y el aspecto que tendrá su contenido de texto.

Como utilizar una pila de fuentes

En este caso, la propiedad font-family tiene múltiples valores. Es una lista priorizada, separada por comas, de nombres de familias de fuentes que puede aplicar al texto, indicando que todas las fuentes son alternativas. Esto permite la máxima compatibilidad con los navegadores y sistemas operativos. La lista se prioriza de izquierda a derecha, de mayor a menor prioridad.

Al aplicar más de un nombre de familia de fuentes, se crea un orden de preferencia. Empezamos con la fuente que queremos primero.
Si un usuario no tiene la primera opción instalada en su ordenador o si no es compatible con el navegador, este pasa a la segunda fuente y la utiliza. Si esa fuente tampoco está disponible, pasa a la tercera, y así sucesivamente.
Podemos listar tantas fuentes como queramos, pero la mejor práctica es listar de tres a cuatro.
Si todo lo demás falla, siempre habrá una fuente genérica listada al final como última opción-mecanismo de devolución.

text-decoration-color

La propiedad CSS establece el color de las decoraciones agregadas al texto por .text-decoration-color

El color se aplica a decoraciones, como subrayados, sobrerayados, tachados y líneas onduladas como las que se utilizan para marcar errores ortográficos, en el ámbito del valor de la propiedad.

CSS no proporciona un mecanismo directo para especificar un color único para cada tipo de línea. No obstante, este efecto se puede lograr anidando elementos, aplicando un tipo de línea diferente a cada elemento (con la text-decoration-linepropiedad) y especificando el color de la línea (con text-decoration-color) por elemento.

text-decoration-thickness

El text-decoration-thickness propiedad CSS establece el grosor del trazo de la línea decorativa que se utiliza en el texto de un elemento, como una línea continua, un subrayado o una línea superpuesta.

capitalize Es una palabra clave que convierte la primera letra de cada palabra a mayúscula. 7.-Letter-spacing

¿Que es letter-spacing?

La propiedad CSS establece el comportamiento del espaciado horizontal entre caracteres de texto. Este valor se agrega al espacio natural entre caracteres mientras se representa el texto. Los valores positivos de hacen que los personajes se separen más, mientras que los valores negativos de los acercan.

letter-spacing: normal;

normal El espacio entre letras normal para la fuente actual. A diferencia de un valor de 0, esta palabra clave permite al agente de usuario alterar el espacio entre caracteres para justificar el texto.

length

Especifica un espacio adicional entre caracteres además del espacio predeterminado entre caracteres. Los valores pueden ser negativos, pero puede haber límites específicos de la implementación. Los agentes de usuario no pueden aumentar ni disminuir más el espacio entre caracteres para justificar el texto.
