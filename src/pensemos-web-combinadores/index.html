Combinadores

Se llaman así porqué combinan otros selectores de manera que proporciona una relación útil entre ellos y la ubicación del contenido en el documento.

Estos selectores seleccionan elementos que son descendientes de otros selectores. No es necesario que sean hijos directos.

Hermanos adyacentes

El selector de elementos hermanos adyacentes (+) se utiliza para seleccionar un elemento que se encuentra justo al lado de otro elemento (es decir a el hermano siguente inmediato) en el mismo nivel de la jerarquía.

Hermanos Generales

Se puede utilizar incluso si no son directamente adyacentes, estos selecciona hermanos que van adelante del punto de partida, de 1 o mas elementos.
